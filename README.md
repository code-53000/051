# 家族树图谱工具

一个纯本地的家族亲属关系管理与可视化工具，帮助大家族理清辈分、称呼和亲属关系。

## 特性

- **🔒 纯本地存储**：所有数据保存在浏览器 IndexedDB 中，不上传到任何服务器
- **🌳 可视化图谱**：使用 D3.js 力导向图展示家族关系，支持缩放、拖动、高亮
- **🧬 智能关系推导**：自动识别 18 种亲属关系（父母、子女、配偶、兄弟姐妹、祖孙、叔伯姑舅、堂表亲等）
- **🏠 分支管理**：按房头、代数筛选查看
- **✨ 交互功能**：
  - 点击成员查看详情和亲属关系
  - 双击展开/收起后代分支
  - 高亮显示直系或旁系亲属
  - 支持姓名搜索、按分支和代数筛选

### 技术栈

- **React 18** + **TypeScript** - 界面开发
- **D3.js** - 力导向图渲染和交互
- **Zustand** - 状态管理
- **IndexedDB (idb)** - 本地数据存储
- **TailwindCSS 3** - 样式
- **Vite 5** - 构建工具
- **Lucide React** - 图标

## 使用说明

### 加载示例数据

首次打开应用，点击右上角「数据管理」→「加载示例数据」，可以加载一份李氏家族的示例族谱（5代38人）体验功能。

### 手动创建族谱

1. 点击「添加成员」录入第一个人（通常是家族始祖）
2. 继续添加其他成员
3. 在成员详情面板中添加父母、配偶、子女关系
4. 图谱会自动生成并展示

### 核心操作

| 操作 | 说明 |
|------|------|
| 单击节点 | 选中成员，高亮显示其亲属，右侧显示详情 |
| 双击节点 | 展开/收起该成员的后代分支 |
| 鼠标悬停 | 高亮显示相关亲属 |
| 滚轮 | 缩放图谱 |
| 拖拽空白处 | 平移图谱 |
| 拖拽节点 | 调整节点位置 |
| 点击空白处 | 取消选中 |

### 筛选功能

- **搜索**：按姓名搜索成员
- **分支筛选**：只查看某一房的成员
- **代数筛选**：只查看某一代的成员
- **显示模式**：全部显示 / 仅直系 / 仅旁系

### 数据导入导出

- **导出 JSON**：将所有数据导出为文件备份
- **导入 JSON**：从备份文件恢复数据（支持覆盖或合并模式）
- **清空数据**：删除所有数据

## 支持的亲属关系

- 直系：父亲、母亲、儿子、女儿、祖父、祖母、孙子、孙女
- 旁系：丈夫、妻子、兄弟、姐妹、叔叔、伯伯、姑姑、舅舅、姨妈、侄子、侄女、外甥、外甥女、堂兄/弟/姐/妹、表兄/弟/姐/妹
